▸ Безопасность — закрытый контур

Один из ключевых требований Эльнура — конфиденциальность данных. Все агенты работают в закрытом контуре через корпоративный VPN. Никакого внешнего трафика, никаких прослушек.

Корпоративный VPN: весь трафик агентов идёт через защищённый туннель, без утечек данных
Изоляция: агенты в отдельной сетевой зоне, изолированы от продакшн-систем CityView
Права доступа: принцип минимальных привилегий, только нужные API-доступы под каждого агента
Хранение секретов: ключи в шифрованном хранилище, не в коде, ротация по графику
План обновлений: регламент patch-management, мониторинг уязвимостей зависимостей
Песочница для каждого агента: даже при промпт-инъекции — выход из песочницы невозможен
